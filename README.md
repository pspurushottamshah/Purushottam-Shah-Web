# The Architect of Transformation

An executive-grade portfolio platform showcasing strategic leadership and business transformation expertise.

## Features

- **Interactive Particle Background**: Mouse-reactive particle system
- **Global Impact Ticker**: Animated counters displaying career metrics
- **Project Deep-Dive**: Interactive project cards with team breakdown charts
- **Career Sprint Timeline**: Scroll-triggered animated timeline
- **Skills Visualization**: Animated proficiency bars
- **Certifications Showcase**: Professional credentials and awards
- **Contact Integration**: Contact form and Calendly scheduling

## Tech Stack

- **Framework**: Next.js 15 with App Router
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Charts**: Recharts
- **Deployment**: Vercel

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Customization

Update your personal information, projects, and career timeline in:
```
data/portfolio.json
```

## Deployment

Deploy to Vercel with one click:

1. Push code to GitHub
2. Import project in Vercel
3. Deploy automatically

## Performance

- Optimized for 90+ Lighthouse scores
- Fast page loads with Next.js Image optimization
- Smooth 60fps animations
- Responsive design for all devices
